# Diabetes Risk Predictive Analysis

### Project Overview
Developed a machine learning pipeline to predict diabetes risk using CDC health indicator data.

### Key Results
* **Accuracy:** 85.00% (Random Forest Classifier)
* **Top Predictors:** BMI, High Blood Pressure, and Age.

### Files in this Repository
* `Diabetes_Risk_Predictive_Analysis_CDC.ipynb`: Full Python code and analysis.
* `diabetes_rf_model.pkl`: The trained predictive model (ready for deployment).
* `diabetes_for_powerbi.csv`: Cleaned dataset for dashboard visualization.
